# RealTime Weather ForeCast using IoT

This Project is a mini weather forecast application using iot where the weather data such as windspeed, humidity and temperature can be viewed from 
a weather api and a DHT11 humidity sensor.The collected data then can be used to be shown on an iot application such as blynk.
This make a great interface for having look at the local weather data and a sumilative weather data of a city that we wanted to monitor.
## Features
Recieves weather data from an Weather API.
Recieves weather data from an on board humidity sensor .
The collected values are shown on an oled display.
The values are pushed via virtual pins to an iot application.
Continous Monitoring of the weather data.
Future upgradable

## Requirements
C++/C sharp for working with IDE.
Arduino IDE software
WeatherMap API Subscriptio free.
Nodemcu
Dht11 sensor
Led Display
Blynk IoT application

String httpGETRequest(const char* serverName) {
  WiFiClient client;
  HTTPClient http;
    
  // Your IP address with path or Domain name with URL path 
  http.begin(client, serverName);
  
  // Send HTTP POST request
  int httpResponseCode = http.GET();
  
  String payload = "{}"; 
  
  if (httpResponseCode>0) {
    Serial.print("HTTP Response code: ");
    Serial.println(httpResponseCode);
    payload = http.getString();
  }
  else {
    Serial.print("Error code: ");
    Serial.println(httpResponseCode);
  }
  // Free resources
  http.end();

  return payload;
}
